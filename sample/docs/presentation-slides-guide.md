## 🎯 スライドの特徴

### 発表慣れしていない方向けの配慮

1. **適切な枚数**: 14枚（1枚30-60秒、余裕を持った構成）
2. **シンプルな構成**: 各スライドは情報を絞り、視覚的に分かりやすく
3. **詳細なスピーカーノート**: 各スライドに話す内容を完全収録
4. **時間配分の明示**: ノートに各スライドの想定時間を記載

## 🚀 使い方

### 1. スライドを開く

#### 方法1: ブラウザで直接開く
```bash
open /Users/kai.ko/dev/internship/sample/docs/presentation-slides.html
```

#### 方法2: ダブルクリック
- Finderで `presentation-slides.html` をダブルクリック

### 2. 基本操作

#### スライド移動
- **次のスライド**: 
  - `→`（右矢印キー）
  - `↓`（下矢印キー）
  - `スペースキー`
  
- **前のスライド**: 
  - `←`（左矢印キー）
  - `↑`（上矢印キー）
  - `Shift + スペースキー`

#### 便利な機能
- **スピーカーノート表示**: `S` キー
  - 別ウィンドウで発表メモが表示される
  - 現在のスライドと次のスライドが確認できる
  - 経過時間も表示される

- **スライド一覧**: `Esc` キー
  - 全スライドを俯瞰できる
  - クリックでそのスライドにジャンプ

- **フルスクリーン**: `F` キー
  - フルスクリーンモードに切り替え
  - もう一度 `F` キーで解除

- **ズーム**: `Alt + クリック`（Mac: `Option + クリック`）
  - 特定の部分を拡大表示

## 🎤 発表の流れ（推奨）

### 準備（発表前）

1. **スライドを開く**
   ```bash
   open /Users/kai.ko/dev/internship/sample/docs/presentation-slides.html
   ```

2. **スピーカーノートを表示**
   - `S` キーを押す
   - 別ウィンドウが開く

3. **ウィンドウの配置**
   - メインウィンドウ: 共有用（フルスクリーン推奨）
   - スピーカーノートウィンドウ: 自分だけが見るモニター

4. **リハーサル**
   - 一度通しで練習
   - 時間配分を確認

### 発表中

1. **オープニング（スライド1-2）**
   - 自己紹介
   - 今日の内容を簡単に説明

2. **本編（スライド3-9）**
   - スピーカーノートを見ながら話す
   - 各スライド30-60秒を意識
   - 焦らずゆっくり話す

3. **デモ（スライド10）**
   - 画面共有を切り替え
   - 実際のECサイトを操作
   - 主要機能を実演（3分）

4. **クロージング（スライド11-14）**
   - 感想と学びを共有
   - 今後の展望
   - まとめと締めの挨拶

## 💡 発表のコツ

### 話し方

1. **ゆっくり、はっきり**
   - 早口にならないよう意識
   - 一文ごとに間を取る

2. **スピーカーノートを活用**
   - 全文が書いてあるので安心
   - そのまま読んでもOK
   - 慣れてきたら自分の言葉で

3. **笑顔で**
   - 楽しんで開発した雰囲気を伝える
   - リラックスして話す

### 時間管理

1. **各スライドの時間を意識**
   - スピーカーノートに時間が書いてある
   - タイマーを見て調整

2. **デモは3分以内**
   - 長くなりすぎないよう注意
   - 主要機能だけに絞る

3. **質疑応答用に余裕を**
   - 7分で終わるように調整
   - フィードバック時間を確保

### トラブル対応

1. **スライドが動かない**
   - ブラウザを更新（F5）
   - 別のブラウザで開く

2. **スピーカーノートが出ない**
   - `S` キーをもう一度押す
   - ポップアップがブロックされていないか確認

3. **時間が押している**
   - スライド8（工夫・つまずき）を短縮
   - デモを2分に短縮

4. **時間が余っている**
   - 各スライドでゆっくり話す
   - デモを丁寧に実演
   - 質疑応答を早めに開始

## 📞 トラブルシューティング

### Q: スライドが表示されない
**A**: 以下を確認してください
1. インターネット接続（CDN読み込みに必要）
2. ブラウザのJavaScriptが有効
3. 別のブラウザで試す

### Q: スピーカーノートが表示されない
**A**: 以下を試してください
1. `S` キーをもう一度押す
2. ブラウザのポップアップブロックを解除
3. 別ウィンドウが最小化されていないか確認

### Q: 文字が小さい/大きい
**A**: ブラウザのズーム機能を使用
- 拡大: `Cmd +`（Mac）/ `Ctrl +`（Windows）
- 縮小: `Cmd -`（Mac）/ `Ctrl -`（Windows）

### Q: フルスクリーンにならない
**A**: 
1. `F` キーを押す
2. ブラウザのフルスクリーンモード（`Cmd+Ctrl+F` on Mac）
