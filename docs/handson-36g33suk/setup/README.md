# 環境構築手順

ハンズオンで使用する開発環境の構築手順です。

---

## 🛠️ 使用するツール

ハンズオンでは、以下のいずれかのAIエージェントIDEを使用します：

- **Cursor**（推奨）
- **Windsurf**（Cursorの代替案）

### どちらを選ぶべきか？

#### Cursorを選ぶ場合

- ✅ 最も人気のAIエージェントIDE
- ✅ 豊富な機能とコミュニティ
- ✅ 無償枠あり（月間使用量に制限あり）

#### Windsurfを選ぶ場合

- ✅ Cursorの無償枠を使い切った場合の代替
- ✅ 無償枠が充実
- ✅ Cascade AIという独自機能
- ✅ Cursorと同様の操作感

**基本的にはCursorを推奨しますが、無償枠を使い切っている場合はWindsurfを選択してください。**

## ✅ 事前確認事項

インストール前に以下を確認してください：

1. **PC環境**
   - OS（Windows / Mac / Linux）
   - OSバージョン
   - 利用可能なブラウザ

2. **ネットワーク環境**
   - インターネット接続環境の有無
   - **外部API（FakeStoreAPI）へのアクセス可否**
     - 企業ネットワーク等で制限される可能性があります
     - 事前に https://fakestoreapi.com/products にアクセスできるか確認してください

3. **アカウント情報**
   - GitHubアカウントの有無（任意）
   - Googleアカウントの有無（任意）
   - Cursor/Windsurfアカウントの有無（当日作成でもOK）
   - Cursor無償枠の利用状況
